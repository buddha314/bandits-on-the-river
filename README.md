# automate-fvtt-pathfinder

Foundry VTT module starter content for Pathfinder rules support in the [`automate-fvtt`](https://github.com/buddha314/automate-fvtt) ecosystem.

## Install by manifest
In Foundry VTT, install this module by manifest URL:

`https://raw.githubusercontent.com/buddha314/automate-fvtt-pathfinder/main/module.json`

## Repository purpose
This repository is structured as an installable Foundry VTT module (manifest at `/module.json`) and provides starter scaffolding for Pathfinder crafting and Kingdom Building content.
