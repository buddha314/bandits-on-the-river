# Open Game Content Designation

This project uses the Open Game License v1.0a in alignment with Pathfinder 1e Open Game Content practices as published on d20pfsrd.com.

## Designation of Open Game Content
The game mechanics and rules text in this repository are designated Open Game Content, except for material designated as Product Identity below.

## Product Identity
The following are designated Product Identity and are not Open Game Content: project branding, logos, module title formatting, and any contributors' names used as marks.
