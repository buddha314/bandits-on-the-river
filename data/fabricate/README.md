# data/fabricate/

Fabricate **crafting-system export JSON** this module provides. Import via
Fabricate's *Import Crafting System* dialog, or seed programmatically through
automate-fvtt's `api.fabricate.seedSystem(...)`.

## riverbend-crafting.json
The 5-item starter (Torch, Rope, Alchemist's Fire (Lesser), Antidote (Lesser),
Healing Potion (Minor)). Results are real **pf2e** items (referenced by
`sourceItemUuid` from `equipment-srd`); recipes are **result-only** for the first
pass (the basic-PF2e ½-Price material cost gets layered in next).

It was generated by `../tools/gen-fabricate-system.mjs`, which drives Fabricate's
**own** `Recipe` model (from the rc.87 source) so the recipe shape is exact, and
validates the payload with Fabricate's `CraftingSystemExporter.validateImportData`
(passes clean). To regenerate/expand to all 10 items, edit the `STARTER` list in
that script (it needs a checkout of the Fabricate rc.87 source — see the path at
the top of the file) and re-run, or author/extend in Fabricate's UI and
*Export System*. Source list: `data/crafting/items.json` and issue #4.
